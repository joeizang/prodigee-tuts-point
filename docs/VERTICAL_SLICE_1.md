# Vertical Slice 1

Vertical Slice 1 is the first committed build target. It is intentionally substantial because the product must prove the real learning loop and content quality bar immediately.

## Platform

- .NET 10 ASP.NET Core backend
- React, TypeScript, Vite frontend
- EF Core with SQLite
- Hybrid content files plus SQLite index/state
- Three themes: light, dark, neutral
- Local learner profile, no authentication
- Manual export/import
- Basic personal notes
- Basic local search
- Basic command palette
- Keyboard shortcuts
- Basic active time tracking

## Curriculum

- C# first
- `wordfreq-csharp`
- Milestone 1: Pure Word Counting Core
- Six rich lessons
- Ten focused exercises
- Visible and hidden tests
- Progressive hints
- Model solutions
- Source reference metadata
- Content quality validator

## Coding

- Monaco multi-file workspace
- Real generated .NET projects per attempt
- Full C# IntelliSense up front
- Backend C# test runner
- Static analysis feedback

## Review and Mastery

- Evidence-based mastery statuses
- Soft locks
- Light diagnostic
- Basic review cards
- Restrained gamification
- Tests plus static analysis plus AI review
- OpenAI-compatible provider abstraction
- Hosted OpenAI and Local Ollama presets

## Acceptance Standard

Vertical Slice 1 is complete only when the learner can:

1. Open the app and use all three themes.
2. Take the light C# diagnostic.
3. Navigate to `wordfreq-csharp`.
4. Study the six-lesson milestone theory cluster.
5. Complete the ten focused exercises in Monaco with full C# IntelliSense.
6. Run visible and hidden tests.
7. Use progressive hints and inspect model solutions.
8. Complete the milestone project task.
9. Receive test, static analysis, and AI review feedback.
10. See progress, mastery state, review cards, notes, search, streak/progress indicators, and study time reflected in the app.
