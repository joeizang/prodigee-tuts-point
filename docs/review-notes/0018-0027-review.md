## Cross-cutting observations

1. **🔴 A real, learner-facing correctness defect exists (0024).** The capstone `run-logquery-summary` model solution depends on `LogQuery.ParseMany`, an unimplemented stub with no exercise/lesson/solution — the published reference solution throws `NotImplementedException`. This single gate is the highest-leverage fix in the project.
