# Tests, Static Analysis, And AI Review For Project Validation

Project validation will include automated tests, static analysis, and AI-assisted review rather than relying on tests alone. This increases first-version complexity, but it matches the product goal: project work should prove correctness, maintainability, design judgment, and production readiness, not merely pass a narrow test suite. AI review will use an OpenAI-compatible provider abstraction so hosted OpenAI and local Ollama-style endpoints can both be supported where their capabilities fit the review workflow.
